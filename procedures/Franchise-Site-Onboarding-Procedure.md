## 1. Purpose

Define steps to onboard a Powered by GridSite or GridColo Inside location to meet technical and security
baselines and enable NOC integration.

## 2. Roles

Franchise Ops, Facilities, Security Engineering, NOC, Legal, ISMS.

## 3. Prerequisites

Executed franchise agreement; site drawings; network plan; access portal/kiosk hardware; Microsoft Teams AV link
for live NOC bridge.

## 4. Procedure

1. Design Review: Validate against Franchise Technical Baseline; confirm segmented networks and secure ingress.
2. Physical Security: Install and enroll access portal and kiosk (badge/face/PIN), guard terminal fallback, escort
   limits, signage, and CCTV coverage; test visitor flow.
3. Identity & Access: Configure roles in portal; background checks for vendor/facility roles before unescorted access;
   establish log forwarding or manual export plan.
4. NOC Integration: Commission resilient AV link and alerting; register site in monitoring and DCIM; define surge
   response roles and escalation.
5. Acceptance: Run checklist; capture evidence; schedule periodic audits.

## 5. Records & Evidence

Checklists, photos, access logs, CCTV coverage map, AV link tests, monitoring/DCIM registrations.

## 6. Metrics

Onboarding cycle time, baseline deviations, evidence completeness.
